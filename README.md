# Python Automation Projects

This repository contains beginner Python automation programs created to practice programming logic and scripting.

## Projects Included

### Calculator Program
A simple calculator that performs arithmetic operations like addition, subtraction, multiplication, and division.

### Car Program
Python program demonstrating object-oriented programming concepts using a car model.

### Loan Eligibility Checker
Program that checks loan eligibility based on conditions like income and age.

### Stopwatch Program
A simple stopwatch application that tracks time using Python.

## Technologies Used
- Python
- Basic Programming Logic
- Functions and Loops
- VS Code
